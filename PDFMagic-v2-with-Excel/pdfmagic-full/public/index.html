⚙️ Extraction Options
How it works: The server extracts text using pdf-parse with a custom renderer that preserves X/Y coordinates to reconstruct rows and columns. Smart heuristics then detect table structures, apply numeric formatting, and build a structured .xlsx workbook with headers, alternating row colors, auto-fit columns, and frozen header rows.
📌 Best results with: PDFs that contain real text (not scanned images), tables with clear column separation (2+ spaces between columns), and structured reports or data exports. Each detected table becomes its own sheet. An Overview sheet is always included.
